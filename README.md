# Odin Project - Mexico Recipe Page

A simple HTML/CSS project built as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum, specifically the CSS Foundations section.

## Project Overview

This project demonstrates the application of CSS fundamentals to style a basic HTML page. The goal is to practice core CSS concepts covered in the Foundations course.

## Skills Practiced

- CSS selectors (class, id, type, combinators)
- CSS properties (color, font, background, etc.)
- The box model (margin, border, padding, content)
- Linking an external CSS stylesheet to an HTML file
- Cascade and specificity

## Built With

- HTML5
- CSS3

## Getting Started

To view the project locally, clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/efrainochoa/odin-mexico.git
cd odin-mexico
open index.html
```
